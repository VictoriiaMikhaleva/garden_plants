# Garden Plants · Уличные растения

Продолжение каталога [Choose_your_plant](https://github.com/VictoriiaMikhaleva/Choose_your_plant) — подбор **уличных** растений по условиям участка.

Каталог по цвету · в соавторстве с ландшафтным дизайнером Анной Колесовой (8 цветовых групп, 139 позиций).

## Что внутри

- `index.html` — лендинг с быстрыми сценариями участка
- `garden_catalog.html` — интерактивный каталог с оценкой совпадения
- `assets/css/` — те же стили, что в комнатном каталоге (`styles.css`, `catalog.css`)
- `assets/js/garden-plants-data.js` — данные растений из PDF
- `assets/js/garden-catalog.js` — логика подбора, избранное, экспорт

## Параметры подбора

| Шкала | Иконка | Описание |
|-------|--------|----------|
| Освещённость | ☀ | 1 (тень) — 5 (полное солнце) |
| Высота | 📏 | 10–200 см |
| Цветение | 🌸 | месяц 1–12 |
| Цвет | фильтр | белые, голубые, фиолетовые, жёлтые, оранжевые, красно-оранжевые, красные, розовые |

## Фото растений

Свои снимки — в `assets/plants/raw/{id}.png`, затем `npm run photos:normalize`.  
Пока фото нет, в карточках цветовые иконки. Подробнее — `assets/plants/PHOTOS_SOURCES.md`.

## Быстрый старт

1. Откройте `index.html` в браузере.
2. Выберите сценарий (тень, полутень, солнце, сезон цветения).
3. В каталоге настройте шкалы и цветовую группу.

## Связь с комнатным каталогом
