# Свои фото для карточек

Wikimedia-фото отключены. Сюда кладите **исходники** перед нормализацией.

## Именование

```
1.png    — Ландыши
2.jpg    — Нарцисс
…
139.jpg  — Пион (розовая группа)
```

Номер = поле `id` в `assets/js/garden-plants-data.js`.

## Рекомендации для генерации (единый стиль)

Как в комнатном каталоге PlantFit:

- **Фон:** чистый белый (#FFFFFF)
- **Ракурс:** растение целиком, вертикально, по центру
- **Пропорции:** близко к 4:3 (или чуть выше — скрипт вписывает в 800×600)
- **Без:** текста, водяных знаков, горшков с брендингом, коллажей

Пример промпта для нейросети:

> Фотореалистичное уличное растение «{название}» в полный рост, цветущий экземпляр, изолировано на чистом белом фоне, мягкий студийный свет, без теней на фоне, 4:3, каталожная съёмка для садового питомника.

## Сборка в каталог

```bash
# положили raw/1.png, raw/2.png …
npm run photos:normalize

# только часть
npm run photos:normalize -- --ids 1,2,3
```

Скрипт: белый фон, кроп 4:3 → `assets/plants/{id}.webp` и поле `photo` в данных.

Старые commons-файлы в `assets/plants/*.webp` можно удалить вручную — каталог на них больше не ссылается после `npm run photos:reset`.
