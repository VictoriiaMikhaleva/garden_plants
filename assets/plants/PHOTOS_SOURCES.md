# Фото растений

## Текущий режим: свои снимки

Автоподбор с Wikimedia **отключён** — качество для каталога не подходило.

Пока нет своих файлов, в карточках показываются **цветовые иконки** (по группе цвета).

### Как добавить фото

1. Сгенерируйте или отснимите исходники → `assets/plants/raw/{id}.png` (см. `raw/README.md`).
2. `npm run photos:normalize` — единый вид: белый фон, 4:3, WebP.
3. Обновите страницу каталога (Ctrl+F5).

### Команды

| Команда | Действие |
|---------|----------|
| `npm run photos:normalize` | Собрать из `raw/` в каталог |
| `npm run photos:reset` | Убрать `photo` из данных (иконки вместо снимков) |
| `npm run photos:commons` | Старый скрипт Wikimedia (не рекомендуется) |

Архив атрибуции commons (если остался): `attribution.json`.
